Beyond Project

Beyond is a career guidance platform designed for high school students to explore university courses, career paths, and alternative opportunities across sectors such as agriculture, education, engineering, health sciences, and more

Team roles

1.Project Manager

Primary Focus: Coordination, integration, and project oversight

Key Responsibilities:

  • Create and maintain project timeline
  • Coordinate daily stand-up meetings
  • Integrate different components
  • Resolve blockers and dependencies
  • Quality assurance and testing
  • Client/stakeholder communication

Tools Needed:

  • Trello/Asana for task management
  • Slack/Discord for communication
  • GitHub for version control
  • Google Docs for documentation

To Familiarize With:

  • Git and GitHub workflows
  • Basic understanding of all technologies used
  • Agile development methodologies

2.Frontend Developer

Primary Focus:User interface and client-side functionality

Key Responsibilities:

  • Implement responsive UI based on designs
  • Develop quiz functionality with JavaScript
  • Ensure cross-browser compatibility
  • Integrate with backend APIs
  • Optimize for mobile devices

Tools Needed:

  • React.js or Vue.js framework
  • CSS preprocessor (SASS/LESS)
  • Chrome DevTools
  • Code editor (VS Code recommended)

To Familiarize With:

  • Modern JavaScript (ES6+)
  • CSS Grid and Flexbox
  • RESTful API integration
  • State management patterns

3.Backend Developer

Primary Focus: Server, database, and API development

Key Responsibilities:

  • Set up server environment
  • Design and implement database schema
  • Create RESTful APIs for frontend
  • Implement user authentication
  • Integrate job search APIs

Tools Needed:

  • Node.js with Express
  • MongoDB or PostgreSQL
  • Postman for API testing
  • Redis for caching (optional)

To Familiarize With:

  • Database design principles
  • API authentication (JWT/OAuth)
  • Server deployment processes
  • Job search APIs (Adzuna, Indeed, etc.)

4.Content Researcher

Primary Focus:Research and content creation

Key Responsibilities:

  • Research career sectors and requirements
  • Develop quiz questions and scoring system
  • Write engaging content for each career path
  • Verify accuracy of all information
  • Create database content structure

Tools Needed:

  • Google Docs/Sheets for organization
  • Notion for content planning
  • Research databases and resources
  • CMS for content management

To Familiarize With:

  • Career information resources
  • Educational content creation
  • Personality assessment methodologies
  • Basic JSON structure for data

5.UX/UI Designer

Primary Focus:User experience and interface design

Key Responsibilities:

  • Create wireframes and prototypes
  • Design user interface components
  • Establish visual identity and style guide
  • Conduct usability testing
  • Ensure accessibility standards

Tools Needed:

  • Figma or Adobe XD
  • UI design toolkit
  • Prototyping software
  • Accessibility checking tools

To Familiarize With:

  • High school user preferences
  • Educational platform design patterns
  • Mobile-first design principles
  • CSS frameworks (Bootstrap, Tailwind)
